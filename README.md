# AgriSmart - Post Harvest Storage Risk Checker

## Problem
Farmers face heavy post-harvest losses due to improper storage conditions like temperature and storage duration.

## Solution
AgriSmart helps farmers check spoilage risk based on:
- Crop type
- Temperature
- Humidity
- Storage days
- space available
- quantity

It provides risk alerts and suggestions to reduce post-harvest losses.

## Features
- Crop selection
- Storage risk prediction
- Simple and user-friendly interface

## Tech Stack
- HTML
- CSS
- JavaScript
